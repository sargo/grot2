GROT²

About

GROT² (eng. arrowhead) is a logic game about making the most valuable chains possible!

Tap or click any field to start a chain reaction - fields will move one by one in the direction shown by arrowheads.

Longer chains grant move bonuses (check multiplier parameter). Clear all fields in a row or in a column to get extra points. Collect as many points as possible!

Competition

You can play this game in demo mode but the fun part starts when you will sign in using GitHub OAuth because all your scores will be included in Hall of Fame. After signing in you can play online or using API.

To simplify development of an API bot you can use GROT² game client. This will help you to focus only on creating the best algorithm to beat other players and their bots and climb to the top of the ranking!

Hall of Fame

To make the results independent of randomness, all players play in the same conditions, that means, in consecutive matches players get the same sequence of arrows. So two players who will choose exactly the same moves or the same algorithm will get the same score.

Hall of Fame is ordered by the average points per match. The rating is updated when you start and finish a match so play all matches to the end to keep high average!
